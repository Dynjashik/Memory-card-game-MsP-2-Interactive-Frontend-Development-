How to play


Match a dog is a win-win memory game with 3 difficulty levels:
    Easy - 8 cards,
    Medium - 12 cards,
    Hard - 16 cards;
The aim of the game is to match all pairs of dog’s breed in the quickest time and minimum number of steps.
Remember, you may flip 2 cards at a time, if they are a match they will remain face up. If they are not you will flip them again, so remember where you left your dogs.
Any time you may restart the game.
New game - new pictures.
Click here to learn more about the different dog’s breeds.
Good Luck!

About


Do you like dogs? Then, this game is for you.

Welcome to a memory matching game Match a dog that will introduce you to different breeds of dogs.

This game has a massive collection of images taken from the Stanford Dogs Dataset.
Follow dog.ceo you can learn more about this project that inspired this game. There you can also add your dog to the collection and support the project in return by receiving a gift for your pet (or small child).
